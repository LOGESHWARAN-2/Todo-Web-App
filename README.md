Task Management System

A simple Task Management System built using React.js for the frontend, Spring Boot for the backend, and MySQL for the database.
This application allows users to create, view, update, and delete tasks efficiently.

Features

User-friendly interface

Create new tasks

View all tasks

Update existing tasks

Delete tasks

RESTful API integration

Tech Stack
Frontend

React.js

HTML, CSS, JavaScript

Backend

Spring Boot

Java

REST APIs

Database

MySQL
